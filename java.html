Text-based Adventure Game

The Text-based Adventure Game is a primitive adventure game that was developed to better understand the principles of object-oriented programming (Encapsulation, Abstraction, Polymorphism and Inheritance). This project was significantly larger (lines of code, number of classes) than its predecessors and as such, introduced me to software engineering practices such as use cases, prototyping and version control (Git).

NB: This project was set as part of CS994: Object Oriented Programming at The University of Strathclyde. For marking purposes the game has a driver class that simulates a playthrough of the game.

Android App: Data Structures & Algorithms

The Android App: Data Structures & Algorithms continues to look at Java but in the context of mobile application development. Here i use Android Studio to develop a mobile application that provides a visual demonstration on how different data structures and algorithms work. This project introduced me to the concept of build tools (Gradle) and how to effectively utilise Git to work remotely as the development process for this project took place during a period of extensive travelling.

Android App: Connect 4

The Android App: Connect 4 project was a collaborative university project consisting of myself and 4 others following an iterative software development life cycle to develop an Android application for a client. The aim of the project was to experience developing as part of a development team and to look at different software development methodologies.
